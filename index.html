Introducción

Esta pagina documenta como se realizo un proyecto escolar. En el cual se diseño el juego del gato (Tic Tac Toe).

Objetivos

  • Crear el juego del gato en Python.
  • Practicar conocimientos de programacion orientada a objetos en Python.

Tecnologias que se usaron.

  • HTML: Para documentar el juego en una pagina web.
  • Python: El lenguaje en el que se desarrollo el juego.
  • GitHub: Para lograr la colaboracion entre los desarrolladores.

Estructura del Juego

El juego del gato se compone de:

  • Un tablero de 3x3 celdas.
  • Dos jugadores: X y O.
  • Un sistema para alternar turnos.
  • Un mecanismo para determinar un ganador o un empate.
  • Un boton para reiniciar el juego en caso de victoria o empate.

Desarrollo:

Python: se importaron las librerias necesarias para

Se importaron las bibliotecas necesarias de Tkinter para construir la interfaz del juego. Se definieron variables que almanenan los botones del tablero y llevan un registro del turno actual. La función check_winner() se encarga de evaluar si hay un ganador al revisar filas, columnas y diagonales del tablero. La función on_button_click() actualiza el botón con el símbolo correspondiente (X u O). La función restart_game() permite reiniciar el juego a su estado inicial. El bucle mainloop() se encarga de mantener la ventana abierta y responder a las interacciones de losjugadores.
